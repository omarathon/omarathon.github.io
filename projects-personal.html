Product Compliance System (Commisioned)

Internal software for the company Red Kite Baby, based in Bristol, England.
  • Manages the safety tests performed on their products, and organises the certificates for such tests.
  • Main application developed in VB.NET, which executes SQL queries to a centralised Microsoft Access database via the Microsoft OLE DB Driver.
  • Uses a Java realtime scheduler, via the Quartz Scheduler library, which sends daily email alerts when certificates are about to expire.

    • Since the main VB.NET application was commisioned software, its code may not be shared publically.
    VB.NET   SQL   Java
