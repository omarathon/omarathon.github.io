    Pong (First Year)

    C programming coursework, for the Computer Organisation and Architecture module. Involved the implementation of the classic "Pong" game on the screen of an oscilloscope via an ARM controller.
  • Worked with a partner, which involved careful planning with a strict deadline.
  • Primitive game logic, such as game loops and rendering.
  • Extensive report detailing and evaluating our result.
  • Our solution involved creating an internal "API" between the oscilloscope screen and the C code, wherein we provided a set of primitive operations such as drawing rectangles and other shapes, which were later combined to form the whole game.

    • Since this code is part of an academic project, it is not available publically.
    Please request access to the private repository via email.
